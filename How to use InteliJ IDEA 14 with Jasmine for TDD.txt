Using Jasmine to test Javascript and possibly PHP (Ali confirm PHP works with Jasmine too)
1) Install InteliJ IDEA 14 (Windows or mac)
2) Go to File > Settings > Languages & Frameworks > Javascript > Libraries
3) Click on Download > Change official Libraries to TypeScript community
4) search for Jasmine (by itself, and also karma-jasmine), click on it and click Download and Install
5) In the Libraries Window, you should see "jasmine-DefinitelyTyped" Global   This is what you need. see pic on this folder. 
6) This will allow you to use Jasmine coding in your Javascript files within InteliJ Idea
7) Run SpecRunner.html
